## 二、AJRG / KING 道場王者 完整 12 欄範例（template）

### 1. 型代碼
- 內部代碼：**AJRG**（攻 A / 術 J / 理 R / 守 G）
- Public Nickname：**KING**
- 化身名：**道場王者**
- 標語：**你是那種打一場就會有人默默截圖上傳的劍士**

### 2. 性格敘述（312 字）

你身上有種讓人心浮氣躁的穩定感。攻得主動、打得規範、想得冷靜、守得扎實。別人試合靠運氣，你試合靠早就寫好的腳本。所謂道場王者不是最猛，是最不出錯。你的動作乾淨到可以當教學影片，你的判斷精準到讓對手懷疑自己練的是不是同一門劍道。

你不追求花俏，追求「該打的有打，不該打的絕不冒險」。師範喜歡你，因為你是活體示範；同門怕你，因為你是那面讓人看清自己不足的鏡子。你不是最受歡迎的人，但你是最受尊敬的人。

你最大的秘密是：你其實很愛贏，只是不屑用髒招。

### 3. 優勢 / 盲點

**優勢**
- 動作完成度高，試合失誤率低
- 壓迫感穩定，對手總是先慌
- 基本功紮實，段審必過

**盲點**
- 過度規範，缺乏出奇制勝的爆發
- 對自己太嚴格，連帶對新人沒耐心
- 太像標準答案，讓人提不起勁學你

### 4. 馆友銳評（道場觀察）

新人看到你第一反應是「天啊這人好帥」，一個月後變成「這人是機器人吧」。你不會主動社交，但所有人私下都想跟你學。你的存在讓道場氣氛多了 10% 的認真度，少了 30% 的放鬆感。是大家心中最像「劍道應該長那樣」的人。

### 5. 服藥指南（修行處方）

你最大的敵人不是對手，是你對完美的執著。偶爾放掉一兩個出頭小手，偶爾打一記不太規範但很爽的胴。你該練的不是更強，是**允許自己不完美**。

### 6. 剋星 / 最佳搭檔

- **剋星**：**BAKA（燃燒中二病）** — 他打不贏你，但他會用亂七八糟的氣合聲和不照腳本的動作把你心態搞爛。你最討厭不按牌理出牌的對手。
- **搭檔**：**SIFU（道場孔明）** — 你們一動一靜，你主攻他主算，組隊幾乎無解。

### 7. 名人對應

- **歷史人物**：**宮本武藏**（二天一流完成期，非晚年五輪書版）— 技術 + 冷靜 + 絕對自信的代表
- **漫畫角色**：**緋村劍心 · 追憶篇** — 技術完成度最高時期的殺手，冷到極致
- **現代代表**：全日本選手權決勝圈裡那種「怎麼打都不犯錯」的教士級劍士

### 8. 職業病診斷

- **最易受傷**：
  - 膝蓋（過度追求規範踏步導致髕骨壓力）
  - 右肘（追求打擊完成度的慢性勞損）
- **預防處方**：每週至少 1 次柔軟度訓練、1 次不穿防具的自由素振釋放肌肉

### 9. 段位演化

| 段位 | 你的狀態 |
|---|---|
| 初段 | 被稱讚「動作漂亮」但被嫌「沒氣勢」 |
| 三段 | 開始有壓迫感，試合勝率顯著拉高，有人開始想跟你對練 |
| 五段 | 變成那個「大家都不想跟你比試合」的人 |
| 七段後 | 可能開道場，成為別人的 SIFU；或轉向研究型，變教練而非選手 |

### 10. 道場戀愛相性

- **最搭**：**OYAJ（道場老爸型）** — 你嚴他暖，道場 CP 模範生
- **最雷**：**SON（熱血少年篇）** — 他演戲你想靜，三天兩頭翻臉
- **18+ 彩蛋**：跟你交往的人會學會怎麼摺劍道衣。這是好事也是壞事。
